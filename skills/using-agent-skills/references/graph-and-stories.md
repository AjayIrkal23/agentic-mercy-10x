# Skill discovery: the graph and the stories

Skills relate as clusters (canonical + aliases + sidecar members). Discover via the skills-index (`hooks/skills-index.json`) and alias map (`hooks/skill-aliases.json`): an alias name resolves to its canonical, and locked skills carry member-of/lead-of cluster links in `hooks/skills-index-overrides.json`. Start from intent, follow the cluster lead, then its references for method depth.
