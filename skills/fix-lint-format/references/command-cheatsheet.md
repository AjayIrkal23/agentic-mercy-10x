# Command cheatsheet (local defaults)

| Context | Typical commands |
| --- | --- |
| Go module + Makefile | `make lint`, `go test ./...`, `go vet ./...` |
| npm | `npm run lint`, `npm run build`, `npm test` |
| pnpm | `pnpm lint`, `pnpm test` |
| yarn | `yarn lint`, `yarn test` |

Always confirm against the repo’s `AGENTS.md`, `README.md`, or CI config when unsure.
